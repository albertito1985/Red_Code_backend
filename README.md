## Books and Quotes Back-end
This is the Back-end from a REST API to handle CRUD functionality for a list of books and quotations.

## Features
- Autentication
- Authorization
- Database handling
- Clean architecture


## Technologies
- C#
- .NET CORE
- Identity
- Entity Framework
- PostgresSQL


## Front-end
The front-end of this app was made with Angularand Typescript and is available at [https://github.com/albertito1985/Red_Code_frontend](https://github.com/albertito1985/Red_Code_frontend/tree/main)
